Here you can request for your website to be indexed.

What is website indexing

To display your website among search results, Mango must first collect some data about it. We mainly collect data about the displayed content.

When to request indexing

You can request website indexing each 30 minutes, but this interval usually isn't necessary. We recommend that you request website indexed when your website has undergone noticable changes in content or structure.

How to request indexing

To request website indexing, simply put the full url to the base of your website in the input field below. The full url to the base of your website is e.g. https://www.mango.com/ and not any other form like https://www.mango.com or mango.com.
